# 產品需求文件（PRD）：HRM DEMO 系統（Python Flask 全站）

## 產品概述
本系統為一個以 Python Flask 開發的 HRM（人力資源管理系統） DEMO 網站，主要展示招聘、員工管理、薪酬與績效三大核心功能。系統設計為單體式網站，資料暫存於記憶體中，不使用 API 或外部資料庫。

## 目標用戶
- 軟體開發學習者：理解 HRM 主要功能與 Flask 開發方式
- 系統展示與測試：快速呈現 HRM 主要模組的功能原型

## 主要功能需求

### 1. 招聘管理（Recruitment Management）
- 顯示現有職缺列表（暫存於記憶體）
- 新增職缺（職務名稱、部門、職缺數量、工作描述）
- 編輯及刪除職缺
- 應徵者模擬申請（輸入姓名、聯絡方式、應徵職缺）

### 2. 員工管理（Employee Management）
- 顯示員工清單（暫存於記憶體）
- 新增員工（姓名、員工編號、部門、職稱、入職日期）
- 編輯及刪除員工資料
- 搜尋員工（按姓名、部門篩選）

### 3. 薪酬與績效管理（Compensation & Performance）
- 員工薪資資料展示及編輯（基礎薪資、獎金、扣款）
- 績效評等記錄（績效等級、評語）
- 計算並顯示實支薪資（基薪加獎金扣款後）

## 非功能需求
- 前端使用 Flask Jinja2 模板渲染，無前後端分離
- 簡潔明瞭的 UI，使用 Bootstrap 或簡易 CSS 樣式
- 資料全部存在伺服器記憶體（使用 Python 全域變數或 Flask session）
- 無需登入驗證，Demo 性質
- 瀏覽器刷新或伺服器重啟後資料重置

## 使用流程範例
1. 使用者進入首頁，看到職缺列表
2. 新增一筆職缺資訊
3. 模擬應徵職缺，輸入應徵者資料
4. 查看員工管理頁面，新增員工資料
5. 編輯薪酬與績效資料，查看計算結果

## 風險與限制
- 資料只存在記憶體，重啟後遺失
- 無登入機制，無法多人同時操作
- 不支援資料永久保存與查詢歷史紀錄

## 未來擴充建議

- 開放 API 支援前後端分離
