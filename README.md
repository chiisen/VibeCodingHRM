# 🏢 VibeCodingHRM
Vibe Coding HRM - 現代化人力資源管理系統 🚀

![HRM01](./images/HRM01.jpg)

---

## 🌟 系統特色與功能模組 (Modules)

1. **🏗️ 專案架構設定**
   - 採用 Flask 建構精簡且強大的應用程式結構
   - 嚴謹的模板與靜態檔案組織架構
   - 完善的依賴套件配置，一鍵輕鬆環境安裝

2. **🎨 基礎 UI 與導航設計**
   - 基於 Bootstrap 5 的全響應式介面，適應各種螢幕尺寸 📱
   - 直覺流暢的跨模組導航體驗
   - 整合 Font Awesome 圖標，大幅提升視覺質感 ✨

3. **🏠 系統總覽首頁**
   - 視覺化系統概覽與即時職缺展示
   - 動態統計數據面板，關鍵數據一眼掌握 📊
   - 貼心的快速功能導航按鈕

4. **🎯 招聘管理模組**
   - 現代化職缺列表（採可展開式分層設計，資訊閱讀更輕鬆）
   - 完整的職缺生命週期管理 (CRUD)
   - 擬真應徵者申請流程模擬
   - 自動化應徵者狀態追蹤與資料管理 📝

5. **👤 核心員工管理**
   - 效率至上的員工資料管理 (CRUD)
   - 智慧搜尋：支援姓名與部門即時篩選 🔍
   - 精準的部門人力分佈統計分析
   - 360 度完整的員工個人檔案視圖

6. **💰 薪酬與績效管理**
   - 全面的薪資組成管理（底薪、獎金、扣款項）
   - 自動化實支薪資即時試算，減少人為錯誤 🔢
   - 視覺化績效評等系統，量化員工表現
   - 深度薪酬成本統計與分析報告

## 🚀 如何快速啟動

1. **📦 安裝依賴套件：**
```bash
pip install -r requirements.txt
```

2. **🔥 啟動系統：**
```bash
python app.py
```

3. **🌐 訪問系統：**
開啟瀏覽器訪問 [http://localhost:5000](http://localhost:5000)

## 📁 專案檔案結構

```bash
VibeCodingHRM/
├── app.py                 # 主應用程式邏輯
├── requirements.txt       # Python 環境依賴
├── PRD.md                 # 產品規格需求文件
├── templates/             # 🎨 HTML 視圖模板目錄
│   ├── base.html         # 全域基礎模板

│   ├── recruitment.html  # 招聘管理介面
│   ├── employees.html    # 員工管理介面
│   ├── compensation.html # 薪酬與績效介面
│   └── ...               # 其他動態表單模板
└── static/               # 📂 靜態資源 (CSS/JS/Images)
```
